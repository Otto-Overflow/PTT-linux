# PTT-linux
A lightweight push-to-talk utility for Linux.

This project listens for configurable keyboard or mouse input and toggles the system microphone accordingly. While the push-to-talk key or button is held, the microphone is unmuted; when released, it is muted again.

The goal is to provide a simple, desktop-agnostic solution that works at the system level, making it compatible with any application that uses the default microphone.

Current status: Linux only. Support for additional platforms may be added in the future.
